# ViroHello

ViroHello is a reversi game implementation using augmented reality thanks to [Viro React](https://viromedia.com/viroreact/) platform.

[![virothello](./src/assets/video-poster.png)](https://vimeo.com/646952573)

## Installation

```sh
npm install
```

## Development

```sh
npm start
```

## Test

```sh
npm test
```
